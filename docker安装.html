有网环境安装

注意事项

  • Docker 运行环境依赖于服务器的内核，它的内核版本要求在 2.6.32-431 以上才可以正常运行。
  • Docker 官方推荐使用的内核为 3.0 以上，在该内核上运行 Docker 可以使服务更加稳定，所以在安装 Docker 之前请将服务器内核升级到 3.0 版本以上。
